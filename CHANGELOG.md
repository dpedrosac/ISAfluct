# Changelog

This project aims to follow [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and [Semantic Versioning](https://semver.org/).

## [2.1] - 2026-02-11

### Changed
- After co-authors comments, the subscores for LID- and hypokinesia-related ISA were introduced.
- Changes in a substanial number of figures performed
- Additional analyses added

### Changed

- Consolidated per-script version notes into this `CHANGELOG.md`. (LLM aided)
